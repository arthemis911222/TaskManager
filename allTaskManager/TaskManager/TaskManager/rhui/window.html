Window

Window继承Panel，并添加了浮动功能。

初始化
方法 描述
Rhui.window(target, options) 创建Window。
target String
jQuery
HTMLElement		 [必填]Window的目标元素，可以是HTML元素、jQuery实例、HTML元素id或者jQuery选择器，如果匹配多个对象，只作用于第一个。
options Object [必填]初始化配置项。
$('#id').rhui('window', options) 在$('#id')元素中创建Window，并返回创建的Window。
options Object [必填]初始化配置项。
Options配置项
选项 类型 默认值 描述
id String undefined 组件实例id，如果不指定则自动生成。
autoShow Boolean true 是否在初始化完毕后自动显示。
modal Boolean true 是否是模态窗口。
content String
jQuery
HTMLElement
Object		 undefined Window的内容。可以是Html文本内容、jQuery实例、Html元素，也可以是远程ajax配置项{url: undefined, iframe: true}。 url是ajax请求路径，iframe表示是否用iframe加载。
hideFooter Boolean false 是否隐藏底部按钮。
buttons Array 底部按钮数组。按钮具体配置请看Toolbar。
expanded Boolean true 是否展开
closeAction String hide 右上角关闭按钮触发的操作，可选项hide和destroy。hide为隐藏，destroy为销毁。
cls String css样式类，多个样式用空格分开。
style Object undefined css样式，会覆盖cls中的相同样式。
bodyCls String Window的body区域css样式类，多个样式类用空格分开。
bodyStyle Object undefined Window的body区域css样式，会覆盖cls中的相同样式。
isBodySize Boolean false 配置项width和height是否为body的高度和宽度。
width Number undefined 宽度，单位像素。
height Number undefined 高度，单位像素
height Number undefined 高度，单位像素
headerTools Array Window右上角按钮。具体配置请看Toolbar。
events Object 事件配置。Loading有 onInit、 onExpand、 onCollapse、 onResize、 onShow、 onHide、 onDestroy事件。
事件
事件 描述
onInit(this) 初始化事件，在Window初始化完成后触发，默认是undefined。
this Window this为当前Window组件。
onExpand(this) Window展开事件，调用expand()方法后触发，默认是undefined。
this Window this为当前Window组件。
onCollapse(this) Window收缩事件，调用collapse()方法后触发，默认是undefined。
this Window this为当前Window组件。
onShow(this) 显示事件，调用show()方法后触发，默认是undefined。
this Window this为当前Window组件。
onHide(this) 隐藏事件，调用hide()方法后触发，默认是undefined。
this Window this为当前Window组件。
onResize(this) 改变大小事件，调用resize()方法后触发，默认是undefined。
this Window this为当前Window组件。
onDestroy(this) 销毁事件，调用destroy()方法后触发，默认是undefined。
this Window this为当前Window组件。
方法
方法 描述
center() 让Window显示在页面中央。该方法返回当前Window，支持链式调用。
moveTo(x, y) 移动Window到指定位置。该方法返回当前Window，支持链式调用。
x Number [必填]x坐标，单位像素。
y Number [必填]y坐标，单位像素。
modeless(val) 设置窗口的模态效果。该方法返回当前Window，支持链式调用。
val Boolean [可选]留空或者true表示非模态，false表示模态。
getTitle() 获取Window标题。
setTitle(title) 设置标题。
getBody() 获取Window的Body元素，返回元素的jQuery实例。
resize(width,height,preventEvent) 重置Window大小，并触发onResize事件。该方法返回当前Window，支持链式调用。
width Number [必填]Window宽度，单位像素。
height Number [必填]Window高度，单位像素。
preventEvent Boolean [可选]true表示阻止onResize事件触发，留空或者false表示允许触发。
setBodySize(width,height,preventEvent) 设置Window的Body大小，并触发onResize事件。该方法返回当前Window，支持链式调用。
width Number [必填]Body宽度，单位像素。
height Number [必填]Body高度，单位像素。
preventEvent Boolean [可选]true表示阻止onResize事件触发，留空或者false表示允许触发。
isExpanded() 判断Window是否展开。
expand(preventEvent) 展开Window，并触发onExpand事件。该方法返回当前Window，支持链式调用。
preventEvent Boolean [可选]true表示阻止onExpand事件触发，留空或者false表示允许触发。
collapse(preventEvent) 收缩Window，并触发onCollapse事件。该方法返回当前Window，支持链式调用。
preventEvent Boolean [可选]true表示阻止事件触发，留空或者false表示允许触发。
load(url,data,done,fail,timeout) 通过ajax方式加载Window内容。该方法返回当前Window，支持链式调用。
url String [必填]ajax请求路径。
data Object [可选]请求参数，如果data数据不为空，则发送post请求。
done Function [可选]请求成功后的回调函数。
fail Function [可选]请求失败后的回调函数。
timeout Number [可选]请求超时，默认60000毫秒。
loadIframe(url) 通过ifram加载Window内容。该方法返回当前Window，支持链式调用。
hide(preventEvent) 隐藏组件，并触发onHide事件。该方法返回当前Panel组件，支持链式调用。
preventEvent Boolean [可选]true表示阻止onHide事件触发，留空或者false表示允许触发onHide事件。
destroy(preventEvent) 销毁组件，并触发onDestroy事件。
preventEvent Boolean [可选]true表示阻止onDestroy事件触发，留空或者false表示允许触发onDestroy事件。
