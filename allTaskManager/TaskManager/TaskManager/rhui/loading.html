Loading

Loading实现加载等待效果，支持整个页面或者指定元素的加载等待。
实例
初始化
方法 描述
Rhui.loading(target, options) 给元素添加Loading效果，返回Loading组件。
target String
jQuery
HTMLElement		 [必填]Loading的目标元素，可以是HTML元素、jQuery实例、HTML元素id或者jQuery选择器，如果匹配多个对象，只作用于第一个。
options String
Object		 [必填]初始化配置项。
Rhui.unloading(target) 取消元素的Loading效果，同时调用destroy()方法销毁Loading组件。
target String
jQuery
HTMLElement		 [必填]Loading的目标元素，可以是HTML元素、jQuery实例、HTML元素id或者jQuery选择器，如果匹配多个对象，只作用于第一个。
$('#id').rhui('loading', options) 给元素添加Loading效果，返回Loading组件。
options String
Object		 [必填]初始化配置项。
Options配置项
选项 类型 默认值 描述
id String undefined 组件实例id，如果不指定则自动生成。
message String 请稍候。。。 显示内容。
cls String css样式类，多个样式用空格分开。
style Object undefined css样式，会覆盖cls中的相同样式。
events Object 事件配置。Loading有 onShow、 onHide、 onDestroy事件
事件
事件 描述
onShow(this) 显示事件，调用show()方法后触发，默认是undefined。
this Loading this为当前Loading组件。
onHide(this) 隐藏事件，调用hide()方法后触发，默认是undefined。
this Loading this为当前Loading组件。
onDestroy(this) 销毁事件，调用destroy()方法后触发，默认是undefined。
this Loading this为当前Loading组件。
onResize(this) 重新计算大小事件，调用setMessage()、resize()方法后触发，默认是undefined。
this Loading this为当前Loading组件。
方法
方法 描述
show(preventEvent) 显示组件，并触发onShow事件。该方法返回当前Loading组件，支持链式调用。
preventEvent Boolean [可选]true表示阻止onShow事件触发，留空或者false表示允许触发onShow事件。
hide(preventEvent) 隐藏组件，并触发onHide事件。该方法返回当前Loading组件，支持链式调用。
preventEvent Boolean [可选]true表示阻止onHide事件触发，留空或者false表示允许触发onHide事件。
destroy(preventEvent) 销毁组件，并触发onDestroy事件。该方法返回当前Loading组件，支持链式调用。
preventEvent Boolean [可选]true表示阻止onDestroy事件触发，留空或者false表示允许触发onDestroy事件。
resize(width,height,preventEvent) 重置大小，居中显示，并触发onResize事件。该方法返回当前Loading组件，支持链式调用。
width Number [必填]宽度，单位像素
height Number [必填]高度，单位像素
preventEvent Boolean [可选]true表示阻止onResize事件触发，留空或者false表示允许触发onResize事件。
setMessage(message,preventEvent) 设置Loading的内容，重新定位，并触发onResize事件。该方法返回当前Loading组件，支持链式调用。
message String [必填]Loading的内容。
preventEvent Boolean [可选]true表示阻止onResize事件触发，留空或者false表示允许触发onResize事件。
