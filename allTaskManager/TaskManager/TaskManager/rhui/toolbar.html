实例
初始化
方法 描述
Rhui.toolbar(target, options) 在target元素中创建toolbar，并返回创建的toolbar。
target String
jQuery
HTMLElement		 [必填]Toolbar的目标元素，可以是HTML元素、jQuery实例、HTML元素id或者jQuery选择器，如果匹配多个对象，只作用于第一个。
options Object [必填]初始化配置项。
$('#id').rhui('toolbar', options) 在$('#id')元素中创建toolbar，并返回创建的toolbar。
options Object [必填]初始化配置项。
Options配置项
选项 类型 默认值 描述
id String undefined 组件实例id，如果不指定则自动生成。
associate Object undefined toolbar的关联组件，该配置项在toolbar的按钮被点击时，会把associate当参数传过去。
tools Array [] toolbar中的按钮数组。请查看Tool配置项。
width Number undefined 宽度，单位像素。
height Number undefined 高度，单位像素
cls String css样式类，多个样式用空格分开。
style Object undefined css样式，会覆盖cls中的相同样式。
events Object 事件配置。Loading有 onShow、 onHide、 onDestroy事件
Tool配置项
选项 类型 默认值 描述
name String undefined 按钮名称，该名称在toolbar中唯一。
type String button 按钮类型，目前有separator和button两种类型。
text String 显示文本。
cls String undefined css样式类，多个样式用空格分开。
click Function undefined 按钮点击事件click(toolbar, associate)。
事件
事件 描述
onShow(this) 显示事件，调用show()方法后触发，默认是undefined。
this Toolbar this为当前Toolbar组件。
onHide(this) 隐藏事件，调用hide()方法后触发，默认是undefined。
this Toolbar this为当前Toolbar组件。
onDestroy(this) 销毁事件，调用destroy()方法后触发，默认是undefined。
this Toolbar this为当前Toolbar组件。
方法
方法 描述
addTool(tool) 添加按钮，返回当前toolbar，支持链式调用。
tool Object [必填]按钮配置项。
addTools(tools) 添加多个按钮，返回当前toolbar，支持链式调用。
tools Array [必填]按钮配置项数组。
removeTool(toolName) 从toolbar中移除按钮，返回当前toolbar，支持链式调用。
toolName String [必填]按钮名称
show(preventEvent) 显示组件，并触发onShow事件。该方法返回当前toolbar组件，支持链式调用。
preventEvent Boolean [可选]true表示阻止onShow事件触发，留空或者false表示允许触发onShow事件。
hide(preventEvent) 隐藏组件，并触发onHide事件。该方法返回当前toolbar组件，支持链式调用。
preventEvent Boolean [可选]true表示阻止onHide事件触发，留空或者false表示允许触发onHide事件。
destroy(preventEvent) 销毁组件，并触发onDestroy事件。该方法返回当前toolbar组件，支持链式调用。
preventEvent Boolean [可选]true表示阻止onDestroy事件触发，留空或者false表示允许触发onDestroy事件。
其他
tool 描述
分隔符|separate 在toolbar中用竖线分隔各个按钮，在tools中用 {type: 'separator'} 添加即可。
