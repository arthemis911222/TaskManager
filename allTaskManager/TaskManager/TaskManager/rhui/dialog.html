方法 描述
Rhui.alert(msg, title, handler) 创建警告框。
msg String [必填]内容。
title String [可选]标题。
handler Function [可选]点击确定按钮时触发的事件。
Rhui.confirm(msg, title, handler) 创建确认框。
msg String [必填]内容。
title String [可选]标题。
handler Function [可选]点击确定或者取消按钮时触发的事件。
Rhui.prompt(msg, title, handler) 创建输入框。
msg String [必填]内容。
title String [可选]标题。
handler Function [可选]点击确定按钮时触发的事件。
